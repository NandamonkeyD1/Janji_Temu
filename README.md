# Janji Temu

Aplikasi berbasis web untuk manajemen janji temu (appointment system).

## 🚀 Fitur Utama
- Manajemen jadwal janji temu
- Sistem booking
- Dashboard admin
- Notifikasi (opsional)

## 🛠️ Teknologi
- PHP (Laravel)
- MySQL
- Bootstrap / UI
